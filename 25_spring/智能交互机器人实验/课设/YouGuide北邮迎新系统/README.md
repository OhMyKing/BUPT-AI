# YouGuide 北邮迎新系统

YouGuide 是面向高校迎新场景的多机器人协同导航与智能交互系统，由机器人端、导览端和管理端组成。系统文档及演示视频位于 `提交材料`，可作为整体设计与功能入口。

- `代码/YouGuideBotSystem`：机器人感知、语音交互、导航与动作控制；
- `代码/YouGuideGuideSystem`：导览业务；
- `代码/YouGuideManagementSystem`：后台管理系统。

项目依赖 ROS、语音/大模型服务及管理端运行环境。真实 API Key 已移除，请复制或参考配置文件，在本地通过环境变量/未跟踪配置注入。模型、公开数据、依赖目录和编译产物未纳入仓库。

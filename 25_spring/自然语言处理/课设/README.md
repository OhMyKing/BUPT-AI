# 自然语言处理课程设计：易卦 AI

易卦 AI 课程设计包含小程序前端、RAG 工具/服务和 BGE 微调实验。结题报告与演示材料位于小组提交目录，代码位于 `YiGuaAISystem`。

仓库未包含训练 checkpoint、模型权重、训练输出和公开语料。需要的数据与模型请从原始来源下载，例如 [Hugging Face BAAI/bge 系列](https://huggingface.co/BAAI)；具体名称和目录以各训练脚本配置为准。Dify 应用可通过项目中的 `.difypkg` 导入。

运行前分别阅读子模块说明并创建隔离环境。不要将 API Key 写入源码或提交本地配置。

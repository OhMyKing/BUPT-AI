# 机器智能课程设计

本目录按实验组织课程设计成果：手写数字识别、八数码问题以及 BlueDon 大语言模型安全检测与增强平台。各实验保留源码、报告、设计文档与演示材料。

训练得到的模型权重、第三方运行环境和公开数据集未纳入仓库。手写数字实验可使用 [MNIST](https://yann.lecun.com/exdb/mnist/)；目标检测实验涉及的 COCO 数据应从 [COCO 官方网站](https://cocodataset.org/#download) 下载。请根据各子项目源码中的路径配置放置数据。

安装依赖时应新建隔离环境，不要提交虚拟环境、模型文件或缓存。API 类实验需自行通过环境变量配置服务密钥。

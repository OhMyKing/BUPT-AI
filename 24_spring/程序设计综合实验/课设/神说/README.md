# 神说（GodSay）

“神说”是程序设计综合实验项目，包含 RPG 游戏主体、账户服务和 FLAP 插件。项目文档位于 `文档`，演示材料位于 `PPT`，源码位于 `程序`。

主要模块：

- `程序/GodSay`：RPG Maker 游戏工程；
- `程序/GodSayAccountManager`：账户管理服务；
- `程序/FLAPplugin`：配套插件；
- `文档`：需求、概要设计和详细设计。

运行前分别依据各模块的 `package.json` 或环境文件安装依赖。ChatGPT 插件不包含真实密钥，需要通过插件参数 `apiKey` 在本地配置。仓库保留演示视频，但不包含 `node_modules`、构建缓存和真实凭证。

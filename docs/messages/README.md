# 留言墙

这里用于收录 BUPT-AI 贡献者想对学弟学妹说的话。内容可以是课程体验、学习方法、课设建议、踩坑记录、大学生活感想，也可以只是一句祝福。

## 如何留言

参与 PR 的贡献者可以在本目录新增一个 Markdown 文件，以姓名、GitHub 用户名或昵称命名，例如：

```text
docs/messages/
├── README.md
├── 王殿云.md
└── OhMyKing.md
```

如果留言包含图片等附件，也可以创建同名子目录：

```text
docs/messages/OhMyKing/
├── README.md
└── images/
    └── campus.jpg
```

留言没有固定模板，可以参考以下结构：

```markdown
# 你的名字或昵称

> 一句话寄语

想对学弟学妹说的话……

— 留言时间（可选）
```

## 基本约定

- 尊重不同经历和选择，友善表达；
- 可以署真实姓名、学号、用户名或匿名；
- 不公开他人的联系方式、账号凭证或高风险敏感信息；
- 不发布广告、攻击性内容或与项目无关的信息；
- 如引用他人文字或图片，请注明来源并确认允许转载。

愿后来的人少踩一点坑，也多看见一些可能性。

# BUPT-Hotel

BUPT-Hotel 是软件工程课程项目，包含酒店管理前端、后端、数据库脚本、课程文档和演示材料。

- `front`：基于 Vite 的前端；
- `backend`：Python 后端与空调控制服务；
- `软件工程`：需求、设计、测试、项目总结等提交文档；
- `视频素材`：系统功能演示。

前端与后端应分别安装依赖。数据库密码、JWT 密钥、天气服务 Key 和大模型 Key 已从仓库版本中移除，请使用环境变量 `DB_PASSWORD`、`JWT_SECRET_KEY`、`VITE_QWEATHER_API_KEY` 和 `VITE_GLM_API_KEY` 配置。

仓库不包含依赖目录、构建产物、嵌套 Git 历史或真实凭证。
